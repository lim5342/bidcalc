# 대장옥션 부동산 종합 계산기

## 프로젝트 개요
경매 낙찰부터 세금·대출·수익까지 한 곳에서 계산하는 부동산 전용 12종 통합 계산기.  
수강생 및 문의 DB 활용 — bosstvauction.co.kr 연동 랜딩 포인트.

---

## 완료된 기능

### 탭별 계산기 (12종)
| 탭 | 기능 | 특이사항 |
|---|---|---|
| 수익률 분석 | 낙찰가→세후 수익률 전과정 | 핵심 탭 |
| 취득세 | 개인/법인/비주거 세율 | 빠른 금액 버튼 |
| 중개수수료 | 매매·전세·월세 법정 요율 | |
| 임대수익률 | 총수익률·순수익률·레버리지 | |
| 전월세전환 | 전세↔월세↔전환율 3방향 | |
| 대출이자 | 원리금균등·원금균등·거치식 | |
| DSR·LTV | 소득기준+담보기준 대출한도 | |
| 증여세 | 관계별 공제·10년 합산 | |
| 상속세 | 배우자 공제·일괄공제 | |
| 종부세 | 1주택·2주택·법인 | |

| 양도세 | 1세대1주택 비과세·다주택 중과 | 빠른 금액 버튼 |

### 수익률 분석 탭 상세
- 나의 현재 상황: 보유주택수(무/1/2/3+) + 소재지(조정/비조정/지방) → 취득세율 자동 설정 + 상황 안내
- 예상 보유기간: 1개월/3개월/6개월/1년/2년/3년/5년 버튼
- 양도세 자동 계산: 보유기간 기준 70%/60%/누진세율 자동, 장기보유공제, 1세대1주택 비과세
- 세전·세후 비교 바 + 게이지 + 등급 배지
- 절세 팁 자동 출력

### 추가된 편의기능 (2026-04-23)
- **한글 금액 자동 표기**: 모든 숫자 입력 시 하단에 "1억 2천만원" 형식 즉시 표시
- **빠른 금액 입력 버튼**: 희망 입찰가·매도금액·취득가·양도가에 5천만~10억 원터치 버튼
- **결과 클립보드 복사**: 상세 분석표 우측 [복사] 버튼 → 텍스트 포맷으로 복사
- **토스트 알림**: 초기화·복사 완료 시 화면 하단 팝업 알림
- **초기화 토스트**: 리셋 버튼 클릭 시 "초기화 완료" 알림
